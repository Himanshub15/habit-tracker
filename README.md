# Bloomkeeper — Habit Garden 🌱

A garden-themed habit tracker where every habit is a living plant. The more consistent you are, the more your garden blooms.

**[Live Demo](https://himanshub15.github.io/habit-tracker)**

---

### Features

- **Garden Metaphor** — Each habit grows through 6 stages: seed, sprout, stem, leaves, bud, and full bloom. Break a streak and watch it wilt.
- **5 Views** — Garden (daily), Weekly grid, Monthly calendar, Year heatmap, and Dashboard with analytics.
- **Smart Scheduling** — Daily, weekdays-only, or custom day selection. Set time-of-day preference (morning/afternoon/evening).
- **Streak Tracking** — Current streaks, longest streaks, completion rates, and goal progress per habit.
- **Categories** — Health, Fitness, Learning, Mindfulness, Productivity, and Custom.
- **Dark / Light Theme** — Toggle with one click, remembers your preference.
- **Zero Dependencies** — Single HTML file. No build step, no framework, no server.
- **Local Storage** — All data stays in your browser. Nothing leaves your machine.

---

### Tech Stack

`HTML` · `CSS` · `JavaScript` · `localStorage`
